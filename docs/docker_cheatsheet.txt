Colinha Docker/Compose - SMC Lab

Status e logs
- docker ps
  Lista containers rodando.
- docker-compose ps
  Status do stack (web, worker, etc).
- docker-compose logs -f web
  Logs em tempo real do servico web.
- docker-compose logs --tail=200 worker
  Ultimas 200 linhas do worker.

Subir/parar
- docker-compose up -d
  Sobe em background.
- docker-compose up
  Sobe em foreground (fecha o terminal, para tudo).
- docker-compose down
  Para e remove containers.
- docker-compose restart web
  Reinicia so o servico web.

Rebuild e atualizacao
- docker-compose up -d --build
  Rebuilda imagens e sobe tudo.
- docker-compose pull
  Baixa imagens definidas com image: no compose.

Comandos dentro do container
- docker-compose exec web bash
  Entra no container web.
- docker-compose exec web python manage.py migrate
  Executa migrations.
- docker-compose exec web python manage.py shell
  Abre shell do Django.
- docker-compose exec worker celery -A trader_portal inspect ping
  Testa se o worker responde.

Limpeza basica
- docker system df
  Mostra espaco usado por imagens/volumes.
- docker image prune
  Remove imagens nao usadas.

Ajuda rapida
- docker-compose --help
- docker --help
